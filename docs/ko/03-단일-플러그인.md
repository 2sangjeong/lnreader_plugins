# 단일 플러그인 만들기

새 소설 사이트를 위한 플러그인을 처음부터 만드는 방법을 설명합니다.

---

## 1단계: 파일 만들기

플러그인 파일은 소설 언어에 맞는 폴더 안에 만듭니다.

```
plugins/
├── english/         ← 영어 소설이면 여기
├── korean/          ← 한국어 소설이면 여기
├── japanese/        ← 일본어 소설이면 여기
└── ...
```

파일 이름은 사이트 이름으로 하고 `.ts` 확장자를 씁니다.

```
plugins/korean/novelsite.ts
plugins/english/nobleMTL.ts
```

---

## 2단계: 아이콘 추가

```
icons/src/<언어>//<플러그인이름>/icon.png
```

예시:
```
icons/src/eng/nobleMTL/icon.png
icons/src/kr/booktoki/icon.png
```

> **반드시 96x96 픽셀**이어야 합니다.

아이콘 경로는 플러그인의 `icon` 속성에서 이렇게 사용합니다:
```typescript
icon = 'src/kr/booktoki/icon.png';
```

## 4단계: 실제 구현하기

### HTML 가져오기

```typescript
// fetchApi를 사용해야 합니다 (일반 fetch 대신)
const response = await fetchApi('https://mysite.com/novels?page=1');
const body = await response.text();
```

### Cheerio로 HTML 파싱하기

Cheerio는 서버사이드 jQuery입니다. HTML에서 원하는 데이터를 쉽게 추출할 수 있습니다.

```typescript
import { load as parseHTML } from 'cheerio';

const $ = parseHTML(body);   // jQuery처럼 $ 사용

// 텍스트 추출
const title = $('.novel-title').text().trim();

// 속성 추출

  const name = $(element).find('.title').text().trim();
  const path = $(element).find('a').attr('href');
  // ...
});

// HTML 내용 가져오기 (챕터 본문에 사용)
const content = $('#chapter-content').html();
```

## 실제 예시: popularNovels 구현

사이트 HTML이 이렇게 생겼다고 가정합니다:
```html
<ul id="novel-list">
  <li>
    <a href="/novel/dragon-hero">
      <img src="https://cdn.mysite.com/covers/1.jpg" />
      <span class="title">드래곤 용사</span>
    </a>
  </li>
  ...
</ul>
```

```typescript
async popularNovels(
  pageNo: number,
  { showLatestNovels }: Plugin.PopularNovelsOptions,
): Promise<Plugin.NovelItem[]> {
  // 정렬 방식에 따라 URL 구성
  const sort = showLatestNovels ? 'latest' : 'popular';
  const url = `${this.site}/novels?sort=${sort}&page=${pageNo}`;

  const response = await fetchApi(url);
  const body = await response.text();
  const $ = parseHTML(body);

  const novels: Plugin.NovelItem[] = [];

  $('#novel-list li').each((_, el) => {
    const name = $(el).find('.title').text().trim();
    const path = $(el).find('a').attr('href');   // "/novel/dragon-hero"
    const cover = $(el).find('img').attr('src');

    if (name && path) {
      novels.push({ name, path, cover });
    }
  });

  return novels;
}
```

## 실제 예시: parseNovel 구현

```html
<!-- 소설 상세 페이지 -->
<h1 class="title">드래곤 용사</h1>
<img class="cover" src="https://cdn.mysite.com/covers/1.jpg" />
<p class="author">작가: 홍길동</p>
<div class="summary">먼 옛날 드래곤이...</div>
<span class="status">연재 중</span>

<ul class="chapter-list">
  <li>
    <a href="/chapter/1">1화 - 시작</a>
    <span class="date">2024-01-15</span>
  </li>
  <li>
    <a href="/chapter/2">2화 - 모험</a>
    <span class="date">2024-01-22</span>
  </li>
</ul>
```

```typescript
async parseNovel(novelPath: string): Promise<Plugin.SourceNovel> {
  const response = await fetchApi(this.site + novelPath);
  const body = await response.text();
  const $ = parseHTML(body);

  // 상태 텍스트를 NovelStatus로 변환
  const statusText = $('.status').text().trim();
  let status = NovelStatus.Unknown;
  if (statusText.includes('연재')) status = NovelStatus.Ongoing;
  else if (statusText.includes('완결')) status = NovelStatus.Completed;

  // 기본 소설 정보
  const novel: Plugin.SourceNovel = {
    path: novelPath,
    name: $('h1.title').text().trim(),
    cover: $('.cover').attr('src'),
    author: $('.author').text().replace('작가: ', '').trim(),
    summary: $('.summary').text().trim(),
    status,
  };

  // 챕터 목록 파싱
  const chapters: Plugin.ChapterItem[] = [];

  $('.chapter-list li').each((index, el) => {
    const link = $(el).find('a');
    const path = link.attr('href');
    const name = link.text().trim();
    const releaseTime = $(el).find('.date').text().trim();

    if (path && name) {
      chapters.push({
        name,
        path,
        chapterNumber: index + 1,
        releaseTime,
      });
    }
  });

  novel.chapters = chapters;
  return novel;
}
```

## 실제 예시: parseChapter 구현

```html
<!-- 챕터 페이지 -->
<div id="content">
  <p>드래곤이 포효했다.</p>
  <p>용사는 검을 들었다.</p>
  <div class="ad-banner">광고</div>  <!-- 제거해야 할 것 -->
</div>
```

```typescript
async parseChapter(chapterPath: string): Promise<string> {
  const response = await fetchApi(this.site + chapterPath);
  const body = await response.text();
  const $ = parseHTML(body);

  // 광고, 불필요한 요소 제거
  $('script, style, .ad-banner, .advertisement').remove();

  // 본문 HTML 반환
  const content = $('#content').html();
  return content || '';
}
```

## 실제 예시: searchNovels 구현

```typescript
async searchNovels(searchTerm: string, pageNo: number): Promise<Plugin.NovelItem[]> {
  // 검색어를 URL 인코딩해서 쿼리 파라미터로 전달
  const url = `${this.site}/search?q=${encodeURIComponent(searchTerm)}&page=${pageNo}`;

  const response = await fetchApi(url);
  const body = await response.text();
  const $ = parseHTML(body);

  const novels: Plugin.NovelItem[] = [];

  $('.search-results .item').each((_, el) => {
    const name = $(el).find('.title').text().trim();
    const path = $(el).find('a').attr('href');
    const cover = $(el).find('img').attr('src');

    if (name && path) {
      novels.push({ name, path, cover });
    }
  });

  return novels;
}
```

## 개발 팁

### 표지 이미지가 없을 때

```typescript
import { defaultCover } from '@libs/defaultCover';

cover: $(el).find('img').attr('src') || defaultCover,
```

### 상대 경로를 절대 경로로 변환

```typescript
// src="/images/cover.jpg" → "https://mysite.com/images/cover.jpg"
const src = $(el).find('img').attr('src') || '';
const cover = src.startsWith('http') ? src : this.site + src;
```

### 공백/개행 문자 제거

```typescript
const text = $(el).text().trim();            // 앞뒤 공백 제거
const text2 = text.replace(/\s+/g, ' ');    // 중간 공백 정리
```
