# 라이브러리 레퍼런스

플러그인에서 `@libs/...`로 import해서 사용하는 유틸리티 함수들을 설명합니다.

---

## `@libs/fetch` - HTTP 요청

### 왜 일반 `fetch` 대신 `fetchApi`를 써야 하나?

`fetchApi`는 자동으로 다음 헤더를 추가해줍니다:
- `Connection: keep-alive`
- `Accept: */*`
- `Accept-Language: *`
- `Sec-Fetch-Mode: cors`
- `Accept-Encoding: gzip, deflate`

또한 LNReader 앱의 프록시를 통해 CORS 문제를 우회합니다.

---

### `fetchApi` - 일반 HTTP 요청

```typescript
import { fetchApi } from '@libs/fetch';

// 기본 사용
const response = await fetchApi('https://example.com/novels');
const html = await response.text();
const json = await response.json();

// 헤더 추가
const response = await fetchApi('https://example.com/api', {
  headers: {
    'Authorization': 'Bearer token123',
    'Referer': 'https://example.com',
  },
});

// POST 요청
const response = await fetchApi('https://example.com/search', {
  method: 'POST',
  body: JSON.stringify({ keyword: '드래곤' }),
  headers: {
    'Content-Type': 'application/json',
  },
});

// FormData POST
const formData = new FormData();
formData.append('search', '용사');
const response = await fetchApi('https://example.com/search', {
  method: 'POST',
  body: formData,
});
```

반환값은 표준 `Response` 객체입니다:
```typescript
response.ok          // 성공 여부 (200-299)
response.status      // HTTP 상태 코드
response.url         // 최종 URL (리다이렉트 후)
await response.text()   // HTML 문자열
await response.json()   // JSON 파싱
await response.arrayBuffer()  // 바이너리 데이터
```

### `fetchText` - 인코딩 지정 요청

한국어, 중국어 등 UTF-8이 아닌 인코딩을 사용하는 사이트에 유용합니다.

```typescript
import { fetchText } from '@libs/fetch';

// EUC-KR 인코딩 사이트
const html = await fetchText('https://old-korean-site.com', {}, 'euc-kr');

// GBK 인코딩 중국 사이트
const html = await fetchText('https://chinese-site.com', {}, 'gbk');

// 기본 (UTF-8)
const html = await fetchText('https://example.com');
```

### `fetchFile` - 파일을 Base64로 가져오기

이미지 등 바이너리 파일을 Base64 문자열로 가져옵니다.

```typescript
import { fetchFile } from '@libs/fetch';

const base64 = await fetchFile('https://example.com/image.jpg');
// → "iVBORw0KGgoAAAANSUhEUgAA..."
```

---

### `fetchProto` - Protocol Buffers 요청

gRPC나 Protobuf API를 사용하는 사이트에 사용합니다. (드물게 사용)

```typescript
import { fetchProto } from '@libs/fetch';

const result = await fetchProto<ResponseType>(
  {
    proto: '...proto 파일 내용...',
    requestType: 'SearchRequest',
    requestData: { keyword: 'dragon' },
    responseType: 'SearchResponse',
  },
  'https://api.example.com/search',
);
```

## `@libs/storage` - 데이터 저장

플러그인 실행 중 데이터를 저장하고 불러올 때 사용합니다. 세션 간에 유지됩니다.

```typescript
import { storage } from '@libs/storage';

// 저장
storage.set('session_id', 'abc123');

// 만료 시간 지정 (10분 후 자동 삭제)
storage.set('cache', data, 10 * 60 * 1000);

// 불러오기
const sessionId = storage.get('session_id');

// 삭제
storage.delete('session_id');

// 모든 키 목록
const keys = storage.getAllKeys();
```

### 실제 사용 예시: 세션 쿠키 캐싱

```typescript
class MyPlugin implements Plugin.PluginBase {
  // 플러그인 로드 시 저장된 값을 바로 읽음
  private sessionId = storage.get('my_session_id') || '';

  async popularNovels(...) {
    // 세션이 없으면 로그인
    if (!this.sessionId) {
      this.sessionId = await this.login();
      storage.set('my_session_id', this.sessionId);
    }

    const response = await fetchApi(url, {
      headers: { 'Cookie': `session=${this.sessionId}` }
    });
    // ...
  }
}
```

### `localStorage` / `sessionStorage`

브라우저의 `window.localStorage`/`window.sessionStorage`에 접근할 수 있습니다. 플러그인이 WebView에서 실행될 때 사이트의 로컬스토리지 데이터를 읽어야 할 때 사용합니다.

```typescript
import { localStorage, sessionStorage } from '@libs/storage';

const data = localStorage.get();  // 모든 localStorage 데이터
```

> **주의**: 웹 테스트 인터페이스(localhost:3000)에서는 동작하지 않습니다. 앱에서만 사용 가능합니다. 이 기능을 사용하는 플러그인은 `webStorageUtilized = true`를 설정해야 합니다.

## `@libs/filterInputs` - 필터 타입

필터를 정의할 때 사용하는 타입과 상수입니다. 자세한 내용은 [필터 시스템 문서](./06-필터-시스템.md)를 참고하세요.

```typescript
import { FilterTypes, Filters } from '@libs/filterInputs';

FilterTypes.Picker              // 드롭다운 선택
FilterTypes.TextInput           // 텍스트 입력
FilterTypes.Switch              // 토글 스위치
FilterTypes.CheckboxGroup       // 체크박스 그룹
FilterTypes.ExcludableCheckboxGroup  // 포함/제외 체크박스
```

## `@libs/defaultCover` - 기본 표지

소설 표지 이미지가 없을 때 사용하는 기본 이미지입니다.

```typescript
import { defaultCover } from '@libs/defaultCover';

novels.push({
  name: '소설 제목',
  path: '/novel/1',
  cover: imageUrl || defaultCover,  // 이미지가 없으면 기본 표지 사용
});
```

## `@libs/isAbsoluteUrl` - URL 판별

경로가 절대 URL인지 상대 경로인지 확인합니다.

```typescript
import { isUrlAbsolute } from '@libs/isAbsoluteUrl';

isUrlAbsolute('https://example.com/image.jpg')  // true
isUrlAbsolute('/images/cover.jpg')               // false

// 활용 예시
const cover = imgSrc
  ? isUrlAbsolute(imgSrc) ? imgSrc : this.site + imgSrc
  : defaultCover;
```

## 외부 라이브러리

플러그인에서 직접 npm 패키지도 사용할 수 있습니다.

### Cheerio - HTML 파싱

```typescript
import { load as loadCheerio } from 'cheerio';

const $ = loadCheerio(html);
$('.title').text();
$('img').attr('src');
```

### dayjs - 날짜 처리

```typescript
import dayjs from 'dayjs';

const date = dayjs('2024년 1월 15일', 'YYYY년 M월 D일').format('YYYY-MM-DD');
// → '2024-01-15'

const relativeDate = dayjs().subtract(3, 'days').format('YYYY-MM-DD');
// → '2024-01-12' (오늘이 2024-01-15라면)
```

### urlencode

```typescript
import { encode, decode } from 'urlencode';

encode('검색어')    // '%EA%B2%80%EC%83%89%EC%96%B4'
decode('%EA%B2%80') // '검'
```

### htmlparser2

대용량 HTML이나 스트리밍 파싱이 필요할 때 사용합니다. Cheerio보다 빠르지만 API가 복잡합니다.

```typescript
import { Parser } from 'htmlparser2';

const parser = new Parser({
  onopentag(name, attrs) { /* 태그 열릴 때 */ },
  ontext(text) { /* 텍스트 */ },
  onclosetag(name) { /* 태그 닫힐 때 */ },
});
parser.write(html);
parser.end();
```
