# 필터 시스템

인기 소설 화면에 "장르", "정렬 순서" 같은 필터 버튼을 추가하는 방법을 설명합니다.

---

## 필터의 동작 방식

1. 플러그인에 `filters` 속성을 정의합니다
2. 앱이 이를 읽어 UI에 필터 버튼을 표시합니다
3. 사용자가 필터를 선택하면 `popularNovels`가 호출될 때 선택한 값이 `options.filters`로 전달됩니다

```
[사용자가 "판타지" 장르 선택]
       ↓
popularNovels(1, { filters: { genre: { value: ['fantasy'] } } })
       ↓
[플러그인이 genre 값을 URL에 붙여 요청]
```

---

## 필터 정의하기

```typescript
import { FilterTypes, Filters } from '@libs/filterInputs';

class MyPlugin implements Plugin.PluginBase {
  filters = {
    // 키 이름 = popularNovels에서 접근할 이름
    order: {
      label: '정렬',                    // 앱에 표시되는 이름
      type: FilterTypes.Picker,         // 필터 종류
      value: '',                        // 기본값 (처음 선택된 값)
      options: [
        { label: '인기순', value: '' },
        { label: '최신순', value: 'latest' },
        { label: '평점순', value: 'rating' },
      ],
    },
  } satisfies Filters;  // ← 반드시 satisfies Filters 추가!
```

> **중요**: `satisfies Filters`를 빠뜨리면 TypeScript 타입 추론이 안 됩니다.

---

## 필터 값 사용하기

```typescript
async popularNovels(
  pageNo: number,
  { filters }: Plugin.PopularNovelsOptions<typeof this.filters>,
) {
  const order = filters.order.value;  // '' | 'latest' | 'rating'

  const url = `${this.site}/novels?sort=${order}&page=${pageNo}`;
  // ...
}
```

## 필터 종류 5가지

### 1. Picker - 드롭다운 (하나만 선택)

```typescript
status: {
  label: '연재 상태',
  type: FilterTypes.Picker,
  value: '',            // 기본값 (첫 번째 option의 value)
  options: [
    { label: '전체', value: '' },
    { label: '연재 중', value: 'ongoing' },
    { label: '완결', value: 'completed' },
    { label: '휴재', value: 'hiatus' },
  ],
},
```

사용:
```typescript
const status = filters.status.value;  // '' | 'ongoing' | 'completed' | 'hiatus'
```

---

### 2. TextInput - 텍스트 입력

```typescript
author: {
  label: '작가 이름으로 검색',
  type: FilterTypes.TextInput,
  value: '',   // 기본값 (빈 문자열)
},
```

사용:
```typescript
const authorName = filters.author.value;  // string
if (authorName) {
  url += `&author=${encodeURIComponent(authorName)}`;
}
```

---

### 3. Switch - 토글 스위치

```typescript
onlyCompleted: {
  label: '완결작만 보기',
  type: FilterTypes.Switch,
  value: false,   // 기본값 (false = 꺼짐)
},
```

사용:
```typescript
const onlyCompleted = filters.onlyCompleted.value;  // boolean
if (onlyCompleted) {
  url += '&status=completed';
}
```

---

### 4. CheckboxGroup - 체크박스 그룹 (여러 개 선택)

```typescript
genres: {
  label: '장르',
  type: FilterTypes.CheckboxGroup,
  value: [],   // 기본값 (선택된 것 없음)
  options: [
    { label: '판타지', value: 'fantasy' },
    { label: '로맨스', value: 'romance' },
    { label: '액션', value: 'action' },
    { label: '학원', value: 'school' },
    { label: '무협', value: 'wuxia' },
  ],
},
```

사용:
```typescript
const genres = filters.genres.value;  // string[] (선택된 값들의 배열)
// 예: ['fantasy', 'action']

if (genres.length > 0) {
  url += '&genres=' + genres.join(',');
}
```

---

### 5. ExcludableCheckboxGroup - 포함/제외 체크박스

체크박스마다 "포함"과 "제외" 두 가지를 선택할 수 있습니다.

```typescript
tags: {
  label: '태그',
  type: FilterTypes.ExcludableCheckboxGroup,
  value: { included: [], excluded: [] },
  options: [
    { label: '이세계', value: 'isekai' },
    { label: '회귀', value: 'regression' },
    { label: '빙의', value: 'possession' },
    { label: '하렘', value: 'harem' },
  ],
},
```

사용:
```typescript
const tags = filters.tags.value;
// tags.included: string[] (포함할 태그들)
// tags.excluded: string[] (제외할 태그들)

if (tags.included.length > 0) {
  url += '&tags_in=' + tags.included.join(',');
}
if (tags.excluded.length > 0) {
  url += '&tags_out=' + tags.excluded.join(',');
}
```

## 실제 예시: 여러 필터 조합

```typescript
class MyPlugin implements Plugin.PluginBase {
  filters = {
    order: {
      label: '정렬',
      type: FilterTypes.Picker,
      value: 'popular',
      options: [
        { label: '인기순', value: 'popular' },
        { label: '최신순', value: 'latest' },
        { label: '평점순', value: 'rating' },
      ],
    },
    status: {
      label: '상태',
      type: FilterTypes.Picker,
      value: '',
      options: [
        { label: '전체', value: '' },
        { label: '연재 중', value: 'ongoing' },
        { label: '완결', value: 'completed' },
      ],
    },
    genres: {
      label: '장르',
      type: FilterTypes.CheckboxGroup,
      value: [],
      options: [
        { label: '판타지', value: '1' },
        { label: '로맨스', value: '2' },
        { label: '액션', value: '3' },
      ],
    },
    showAdult: {
      label: '성인 콘텐츠 포함',
      type: FilterTypes.Switch,
      value: false,
    },
  } satisfies Filters;

  async popularNovels(
    pageNo: number,
    { showLatestNovels, filters }: Plugin.PopularNovelsOptions<typeof this.filters>,
  ): Promise<Plugin.NovelItem[]> {
    // showLatestNovels가 true면 정렬을 latest로 덮어씀
    const order = showLatestNovels ? 'latest' : filters.order.value;

    let url = `${this.site}/novels?sort=${order}&page=${pageNo}`;

    if (filters.status.value) {
      url += `&status=${filters.status.value}`;
    }

    if (filters.genres.value.length > 0) {
      url += `&genres=${filters.genres.value.join(',')}`;
    }

    if (filters.showAdult.value) {
      url += `&adult=1`;
    }

    // ... 이후 fetch 및 파싱
  }
}
```

## 멀티소스 필터 (filters/*.json)

멀티소스 플러그인에서는 각 사이트별로 JSON 파일로 필터를 정의합니다.

```json
// plugins/multisrc/madara/filters/mysite.json
{
  "filters": {
    "genres": {
      "type": "CheckboxGroup",
      "label": "Genres",
      "value": [],
      "options": [
        { "label": "Action", "value": "action" },
        { "label": "Fantasy", "value": "fantasy" },
        { "label": "Romance", "value": "romance" }
      ]
    },
    "status": {
      "type": "Picker",
      "label": "Status",
      "value": "",
      "options": [
        { "label": "All", "value": "" },
        { "label": "Ongoing", "value": "on-going" },
        { "label": "Completed", "value": "end" }
      ]
    }
  }
}
```

파일 이름은 `sources.json`의 `id` 값과 같아야 합니다.

---

## 주의사항

- 필터 `value`는 기본값이자 초기 상태입니다. 신중하게 설정하세요
- `satisfies Filters`를 빠뜨리면 `options.filters.xxx.value`에서 타입 오류가 납니다
- 필터를 추가/삭제할 때는 플러그인 버전의 `minor`를 올려야 합니다
